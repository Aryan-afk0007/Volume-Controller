#  Volume Controller

Control your computer's system volume using **hand gestures** instead of your keyboard or mouse. This project leverages **Computer Vision** and **Artificial Intelligence** to recognize hand movements in real time and convert them into volume control commands.

---

## 📖 Overview

The **AI Gesture Volume Controller** is a Python-based application that uses a webcam to detect hand landmarks with **MediaPipe**. By measuring the distance between the thumb and index finger, the application increases or decreases the system volume, providing a seamless touch-free experience.

This project demonstrates practical applications of **Computer Vision**, **Human-Computer Interaction (HCI)**, and **Gesture Recognition**.

---

## ✨ Features

* 🎥 Real-time webcam hand tracking
* ✋ Accurate hand landmark detection using MediaPipe
* 🔊 Touch-free system volume control
* 📊 Live volume percentage and volume bar
* ⚡ Smooth and responsive gesture recognition
* 🌍 Cross-platform media key support
* 💻 Lightweight and beginner-friendly

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI_Gesture_Volume_Controller.git
```

### 2. Navigate to the project folder

```bash
cd AI_Gesture_Volume_Controller
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the application using:

```bash
python volume_controller.py
```

### Controls

* 🤏 Move your thumb and index finger closer to **decrease** the volume.
* 🤏 Move them farther apart to **increase** the volume.
* Press **Q** to quit the application.
